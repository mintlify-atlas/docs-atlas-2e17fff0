## Why Obsidian Portable?

Obsidian Portable brings the powerful markdown note-taking capabilities of Obsidian to a truly portable format. Built with the PortableApps.com launcher, it allows you to carry your entire knowledge base with you.

<CardGroup cols={2}>
  <Card title="True Portability" icon="usb-drive">
    Run from USB drives, cloud folders, or any removable media without installing on the host PC
  </Card>
  <Card title="Automatic Path Management" icon="route">
    Seamlessly handles vault locations as you move between different drives and computers
  </Card>
  <Card title="Zero Installation" icon="circle-xmark">
    No local installation required — just plug in and run from any Windows PC
  </Card>
  <Card title="Preserves Settings" icon="floppy-disk">
    All your settings, plugins, and data travel with you between computers
  </Card>
</CardGroup>

## Key Features

<AccordionGroup>
  <Accordion title="PortableApps.com Integration" icon="puzzle-piece">
    Built using the PortableApps.com launcher framework for seamless portability and integration with the PortableApps.com Platform.
  </Accordion>
  
  <Accordion title="Smart Data Directory Management" icon="folder-tree">
    Uses `--user-data-dir` command-line argument to keep all Obsidian data in the portable directory structure.
  </Accordion>
  
  <Accordion title="Drive Letter Independence" icon="arrows-rotate">
    Automatically updates vault paths when your portable drive changes letter, ensuring vaults remain accessible.
  </Accordion>
  
  <Accordion title="Multi-Version Support" icon="code-branch">
    Supports the latest Obsidian versions with easy upgrade path for new releases.
  </Accordion>
</AccordionGroup>

## Get Started

<Steps>
  <Step title="Download">
    Download the latest version from the [GitHub Releases](https://github.com/Papela/Obsidian-Portable) page
  </Step>
  
  <Step title="Extract">
    Extract the portable package to your USB drive, cloud folder, or portable location
  </Step>
  
  <Step title="Launch">
    Run `ObsidianPortable.exe` to start using Obsidian with full portability
  </Step>
</Steps>
