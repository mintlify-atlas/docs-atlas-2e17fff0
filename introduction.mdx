# Introduction to Obsidian Portable

Obsidian Portable is a portable distribution of Obsidian, the powerful plain-text markdown note-taking application. Packaged with a PortableApps.com launcher, it allows you to use Obsidian from a USB drive, cloud folder, or local directory without needing to install it on each PC.

## What is Obsidian Portable?

Obsidian Portable takes the full-featured Obsidian markdown editor and packages it as a completely portable application. This means:

- **No installation required** - Run directly from any location
- **Your data travels with you** - All settings and vaults stay in the portable directory
- **No registry changes** - Leaves no trace on host computers
- **Drive letter independent** - Works even when your drive letter changes

<Info>
Obsidian Portable uses the `--user-data-dir` parameter to store all application data within the portable directory structure, ensuring complete portability.
</Info>

## Comparison with Standard Obsidian

| Feature | Standard Obsidian | Obsidian Portable |
|---------|------------------|-------------------|
| Installation | Required | Not required |
| Registry entries | Yes | No |
| User data location | `%APPDATA%` | Portable directory |
| Drive letter changes | N/A | Automatically handled |
| Admin rights | May be required | Not required |
| Leaves PC clean | No | Yes |

## How It Works

Obsidian Portable uses the PortableApps.com launcher framework to:

1. Launch Obsidian with `--user-data-dir` pointing to the portable `Data` directory
2. Automatically update vault paths when the drive letter changes
3. Store all settings, plugins, and themes in the portable directory
4. Clean up temporary files on exit

<Note>
The launcher configuration (`ObsidianPortable.ini`) handles path replacements automatically, updating references in `obsidian.json` when your drive letter changes.
</Note>

## Data Privacy

Your data privacy is paramount:

- All notes are stored locally on your device
- No account creation required for basic use
- No data sent to servers (unless using Obsidian Sync/Publish)
- No telemetry or tracking in the portable launcher
